Crie um aplicativo web React que exibe uma lista de usuários e permite a filtragem por nome de usuário. A lista de usuários deve ser obtida a partir de uma API RESTful fornecida.

O aplicativo deve exibir inicialmente todos os usuários em uma lista paginada. A lista deve mostrar a imagem do perfil, o nome do usuário e uma breve descrição. Clicar em um usuário deve levar o usuário a uma página de detalhes do usuário, que deve exibir mais informações sobre o usuário, como endereço, telefone, e-mail etc.

O usuário deve poder filtrar a lista de usuários por nome de usuário, usando uma barra de pesquisa. A lista de usuários deve ser atualizada em tempo real, conforme o usuário digita na barra de pesquisa.

O aplicativo deve ser responsivo e funcionar em dispositivos móveis e desktops. O código do aplicativo deve ser organizado, modular e fácil de manter. Use o React Hooks e as melhores práticas para desenvolver o aplicativo.

Bônus: Implemente testes de unidade e integração para o aplicativo.
